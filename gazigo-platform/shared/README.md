# @gazigo/shared

Shared utilities, types, branding assets, and constants for the GAZIGO platform.

## Contents

- **Branding**: Colors, theme, constants
- **Types**: TypeScript type definitions
- **i18n**: Internationalization strings (French/English)

## Usage

```typescript
import { brandColors, BRAND, CYLINDER_TYPES } from '@gazigo/shared';
import type { Order, User, Customer } from '@gazigo/shared';
```
